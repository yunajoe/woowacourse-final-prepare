# 편의점

- 구매자의 할일 혜액과 재고 상황을 고려하여 최종 결제 금액을 계산하고 안내하는 결제 시스템 구현

## 기능 요구 사항

1. 결제 금액

- 사용자가 입력한 사움의 가격과 수량을 기반으로 최종 결제 금액을 계산한다.
- 총 구매액은 상품별 가격과 수량을 곱하여 계산하며, 프로모션 및 멤버십 할인 정책을 반영하여 최종 결제 금액을 산출한다.
- 구매 내역과 산출한 금액 정보를 영수증으로 출력한다.
- 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시키고, "[ERROR]" 로 시작하는 에러 메시지를 출력 후 그 부분부터 입력을 다시 받는다.
- Exception이 아닌 IllegalArgumentException 혹은 IllegalStateException 등과 같은 명확한 유형을 처리한다.

2. 재고관리

- 각 상품의 재고 수량을 고려하여 결제 가능 여부를 확인한다
- 고객이 상품을 구매할 때마다,결제된 수량만큼 해당 사품의 재고에서 차감하여 수량을 관리한다
- 재고를 차감함으로써 시스템은 최신 재고 상태를 유지하며, 다음 고객이 구매할 떄 정확한 재고 정보를 제공한다

3. 프로모션 할인

- 오늘 날짜가 프로모션 기간 내에 포함된 경우에만 할인을 적용한다.
- 프로모션은 N개 구매시 1개 무료 증정(Buy N Get 1 Free)의 형태로 진행된다.
- 1+1 또는 2+1 프로모션이 각각 지정된 상품에 적용되며, 동일 상품에 여러 프로모션이 적용되지 않는다.
- 프로모션 혜택은 프로모션 재고 내에서만 적용할 수 있다.
- 프로모션 기간 중이라면 프로모션 재고를 우선적으로 차감하며, 프로모션 재고가 부족할 경우에는 일반 재고를 사용한다.
- 프로모션 적용이 가능한 사움에 대해 고객이 해당 수량보다 적게 가져온 경우, 필요한 수량을 추가로 가져오면 혜택을 받을 수 있음을 안내한다.
- 프로모션 재고가 부족하여 일부 수량을 프로모션 혜택 없이 결제해야 하는 경우, 일부 수량에 대해 정가로 결제하게 됨을 안내한다.

4. 멤버십 할인

- 멤버십 회원은 프로모션 미적용 금액의 30%을 할인 받는다.
- 프로모션 적용 후 남은 금액에 대해 멤버십 할인을 적용한다.
- 멤버십 할인의 최대 한도는 8,000원 이다.

5. 영수증 출력

- 영수증은 고객의 구매내역과 할인을 요약하여 출려한다.
- 영수증 항목은 아래와 같다.
  a) 구매 상품 내역: 구매한 상품명, 수량, 가격
  b) 증정 상품 내역:프로모션에 따라 무료로 제공된 증정 상품의 목록
  c) 금액 정보:
  c-1)총 구매역: 구매한 상품의 총 수량과 총 금액
  c-2)행사 할인: 프로모션에 의해 할인된 금액
  c-3)멤버십 할인: 멤버십에 의해 추가로 할인된 금액
  c-4)내실돈: 최종 결제 금액
- 영수증의 구성 요소를 보기 좋게 정렬하여 고객이 쉽게 금액과 수량을 확인하게 할 수 있게 한다.
